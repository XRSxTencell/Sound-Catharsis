# **App Name**: SoundCatharsis

## Core Features:

- Real-time dB Meter: Display current sound levels in decibels (dB) using the device's microphone.
- Interactive Sound Graph: Visualize sound levels over time with an interactive graph.
- Global Leaderboard: Maintain a global leaderboard displaying the highest recorded dB levels.
- User Profiles: Allow users to create profiles to submit and track their personal best dB levels.
- Data Persistence: Store user profiles and dB records.
- Noise Event Detection: Use a generative AI tool to identify when and where loud sounds originate to create categories of loud sound types (such as loud music or construction).

## Style Guidelines:

- Primary color: Vibrant green (#39FF14) to represent energy and sound spikes.
- Background color: Dark gray (#222222) to enhance visibility and reduce eye strain in a dark theme.
- Accent color: Electric blue (#7DF9FF) for interactive elements and highlights.
- Body and headline font: 'Inter' (sans-serif) for clear readability.
- Use minimalist, outlined icons in electric blue to represent sound-related actions and data.
- A clean, modern layout with a focus on real-time data visualization, ensuring key information is easily accessible.
- Subtle animations and transitions to provide feedback on user interactions and dB level changes.
